## How to clone and setup the project

git clone https://github.com/Abrahamkivosh/freelancer.git

cd freelancer

composer install

npm install

 cp .env.example .env

 create database named " freelancer "

 run 
 php artisan  migrate --seed

 php artisan  serve


 -- You are ready to go
